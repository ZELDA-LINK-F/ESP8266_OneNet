# 工程实训

## 项目简介

本项目为物联网（IoT）综合项目实战课程配套工程，涵盖从 PCB 硬件设计、嵌入式 MCU 固件开发、IoT 通信协议、云平台对接，到上位机可视化界面的完整开发链路。

## 实训周期

- **开始日期**：2026 年 7 月 10 日
- **结束日期**：2026 年 7 月 20 日
- **总时长**：11 天（含 1 天休息日：7 月 18 日）
- **实训形式**：项目驱动，边学边做

## 技术栈概览

| 层级 | 技术内容 |
|------|----------|
| 硬件层 | Altium Designer 20.2.6（原理图 + PCB Layout） |
| 嵌入式层 | MCU 外设（GPIO / 中断 / I2C）、AT 指令框架、传感器驱动 |
| 通信层 | MQTT 协议、阿里云物联网平台 |
| 应用层 | PyQt5 上位机、数据可视化 |
| 辅助工具 | AI 编程助手 |

## 备注

- 原理图和 PCB 文件需使用 **Altium Designer 20.2.6** 打开
- 所有源文件命名建议使用英文或拼音，避免中文路径导致兼容性问题
- Gerber 文件输出前务必自检并完成 DRC
- 文档保存编码：UTF-8（无 BOM），避免中文丢失
